# Express-js
Este repositorio esta enfocado a mejorar mis habiliadades en express para terminar el repositorio de registro de notas

* comando npm init -y : crea un package.json 
* comando npm i nodemon -D: recarga automaticamente los cambios hechos en servidor
* comando npm i ejs: plantilla para pintar datos en html 
* comando npm i morgan: es un middleware que ya trea varias funciones por defecto
